01 电气工程及其自动化

工学门类 / 电气类 / 代码 080601 · 所属一级学科：电气工程

研究电能的产生、传输、变换、控制与应用，是现代工业与能源体系的基石。强电与弱电结合、软件与硬件结合，俗称"工科万金油"，国民经济基础产业专业。

📚 课程体系

基础课（公共基础 + 学科基础）
高等数学线性代数概率论 大学物理复变函数与积分变换电路原理 模拟电子技术数字电子技术工程电磁场 C 语言程序设计工程力学
专业核心课
电机学电力电子技术电力系统分析 电力系统继电保护高电压技术自动控制原理 信号与系统电力拖动与控制系统发电厂电气部分 电力系统自动化
方向课：电力系统 / 电力电子与电力传动 / 高电压 / 电机电器 / 新能源

02 机械设计制造及其自动化

方向课：机械设计 / 机械制造 / 机电一体化 / 智能制造 / 车辆工程
